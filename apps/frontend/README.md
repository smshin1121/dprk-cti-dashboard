# Frontend Scaffold

Vite + React + TypeScript 기반 SPA 준비 상태다. shadcn/ui, Tailwind, Zustand, TanStack Query를 위한 구조만 먼저 맞춘다.

## Included

- Vite React TS entrypoint
- Tailwind baseline
- Zustand store placeholder
- TanStack Query provider
- API/LLM proxy endpoint env wiring
